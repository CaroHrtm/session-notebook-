# GitHub Verknüpfung für Anfänger

## lokales Repo erstellen (in Terminal)

1. Folder für Repo erstellen: `mkdir foldername`
2. Repo im folder erstellen:`git init`
3. Änderungen vornehmen (z.B. Struktur erstellen, Datei erstellen, Inhalt hinzufügen)
4. Änderungen stagen, `git add`eingeben
5. Änderungen commiten `git commit -m "Titel für die Änderung"` eingeben

## remote Repo erstellen (auf GitHub)

1. Profil-logo -> repositories
2. "New" repo erstellen
3. repository name (gleicher Name wie "foldername (s. oben)")
4. "create repository" bestätigen
5. "SSH" anklicken
6. "push an existing repository"- Inhalt kopieren und im Terminal einfügen und bestätigen.
